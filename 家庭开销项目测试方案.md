# 💰 实用版学习计划：家庭开销记录器

## 项目概述

**家庭开销记录器 (Family Expense Tracker)** - 真正日常会用的应用

### ✅ 为什么选择这个实用版本？

- 💯 真实需求：每个人都要管理家庭开支
- 📱 日常使用：每天都会用到
- 📊 直观可见：花费分类和统计图表
- 💵 实用价值：真正帮助理财
- 🎯 简单易用：功能简单，逻辑清晰
- ✅ 保持简单：仍然是4个核心操作

## 💡 项目核心价值

帮助家庭合理管理日常开支，记录每一笔花费，分类统计，让理财变得简单。

## 🎯 核心功能（简单实用）：

1. ➕ 记录开支（例如："超市购物 - 150元"）
2. 📝 编辑开支记录
3. 🗑️ 删除开支记录
4. 👀 查看开支列表和统计
5. 📊 查看分类统计图表（食物、交通、娱乐等）

## 📊 开销分类（符合日常习惯）：

- 🍽️ 食物餐饮
- 🚗 交通出行
- 🏠 住房水电
- 👕 服装购物
- 🎉 娱乐休闲
- 💊 医疗健康
- 📚 学习教育
- 🎁 其他杂项

## 🛠️ 技术栈

- **前端：** Next.js 15 + TypeScript
- **UI：** Tailwind CSS + shadcn/ui
- **图表：** Chart.js / Recharts
- **数据库：** PostgreSQL (Neon)
- **ORM：** Drizzle ORM
- **状态管理：** Zustand
- **部署：** EdgeOne Pages

## 📊 实用功能展示

### 用户日常使用流程：

1. **记录开支** → 点击"添加"，输入金额、选择分类、备注
2. **查看列表** → 滚动浏览开支历史
3. **查看统计** → 底部图表显示分类占比
4. **分析消费** → 月度统计帮助理性消费

### 核心价值：

- 📈 **可视化消费：** 清晰看到钱花在哪里
- 📅 **月度对比：** 本月 vs 上月，对比消费习惯
- 🎯 **分类管理：** 不同类别的开支占比
- 💡 **理性消费：** 通过数据反思消费行为
